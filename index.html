Travel Journal

Travel Journal is a simple MERN (MongoDB, Express.js, React, Node.js) web app that allows users to create and share their travel experiences. The app features a user authentication system with a login and sign up page. Users can create new entries by filling out a form which includes a date picker, place name, and a description of their experience. Users can also upload an image of the place they visited which is stored on the local environment using Multer. Each entry can be clicked on to view the full description and image uploaded. It's a perfect tool for keeping track of your travel memories and sharing them with others.
